Conclusão

\(I)\) Quando se utiliza o Recipiente \(R_1\) (saco plástico pequeno) não há diferençaa significativa no desenvolvimento das mudas das \(2°\) Espécies;

\(II)\) Quando se utiliza o Recipiente \(R_2\) (saco plástico grande) há diferença significativa no desenvolvimento das mudas das \(2°\) Espécies, sendo melhor para a Espécie \(E_1\) (Eucalyptus citriodora);
